关键指标对美债期货影响参考

经济指标数据强于预期数据弱于预期影响逻辑
非农就业 (NFP)利空利多就业强劲 → 加息预期 → 债价下跌
CPI 消费者物价利空利多通胀高企 → 加息 → 收益率升 → 债价跌
PPI 生产者物价利空利多生产成本升 → 通胀传导 → 利空债市
GDP 国内生产总值利空利多经济增长 → 风险偏好升 → 资金流出债市
失业率利多利空失业升 → 降息预期 → 债价上涨
美联储利率决议视具体情况加息=利空 / 降息=利多 / 鹰鸽措辞
ISM 制造业PMI利空利多制造业扩张 → 经济向好 → 减少避险
零售销售利空利多消费强劲 → 经济过热 → 利空债市
初请失业金利多利空申请增加 → 劳动力疲软 → 降息预期
消费者信心指数利空利多信心高涨 → 消费/通胀升 → 利空债市
